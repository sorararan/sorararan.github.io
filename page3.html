gitメモ
使いそうなコマンド集
  • ローカルコンピュータで作ったリポジトリの初期化
    • git init
  • リポジトリの状態を確認
    • git status
  • ステージ領域へファイルを追加(ファイル名のところを.にするとフォルダ全部できる)
    • git add ファイル名
  • リポジトリの歴史を記録
    • git commit -m コメント
  • コミットログを確認
    • git log
  • 変更差分を確認
    • git diff
  • ブランチを一覧表示
    • git branch
  • ブランチ作成
    • git branch ブランチ名
  • ブランチの移動
    • git checkout ブランチ名
  • ブランチのマージ
    • git merge --no-ff ブランチ名
  • 歴史を戻る
    • git reset --hard コミットID
  • リモートリポジトリの登録
    • git remote add 登録名 リポジトリのパス(githubのページでコピーできるあれ)
  • リモートリポジトリへ送信
    • git push
  • リモートリポジトリを取得
    • git clone リポジトリのパス(あれ)
よくあるパターン(?)
 git branch ファイル名でブランチで分ける(ローカル環境内でmasterがバックアップ、ミスったらmasterへ帰る)
↓
git checkout ブランチ名で作ったブランチへ移動
↓
実作業が上手くいった
↓
git merge --no-ff ブランチ名で作業してるところをmasterにしちゃう
↓
git add ファイル名でファイル追加
↓
git commit -m ファイル名でコミット
↓
git pushで自分のgithubにアップロード
↓
github上で元々のリポジトリにプルリクエスト送信
↓
他の人にマージしてもらう
