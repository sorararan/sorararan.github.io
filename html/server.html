サーバ関連メモ
ssh関連
  • ログインするとき
    • ssh ユーザ名@アドレス (-p ポート番号)

  • RSA鍵の作り方
    • ssh-keygen -t rsa
    対話式でファイル名、パスワード、パスワード(確認)を入力
    これで秘密鍵と公開鍵ができる、サーバ上の公開鍵とクライアント上の秘密鍵で認証ができる

  • ファイルを他の端末に送る
    • scp 送りたいファイル名 ユーザ名@アドレス:パス
    (フォルダを送りたい時はscpの後に-rオプション)

  • 上で鍵を作って公開鍵をサーバに送ったら
    • cat id_rsa.pub >> ~/.ssh/authorized_keys

mysql関連
  • mysqlに入るとき
    • mysql -u ユーザ名 -p
    でパスワードが聞かれるから書いてログイン

  • ユーザ作成
    • create user 'ユーザ名'@'ホスト名' identified by 'パスワード';

  • 権限
    • grant 権限 on データベース名.* to 'ユーザ名'@'ホスト名';
    (例えば、全ての権限はall privilegesとか)

  • データベース作成
    • create database データベース名;

  • データベースとかテーブル一覧
    • show databases;
    show tables;

  • データベース選択
    • use データベース名;

  • データベースの削除
    • drop database データベース名;

  • テーブル作成
    • create table テーブル名(カラム名 データ型, ..., プライマリキー);

  • 挿入
    • insert into テーブル名(カラム名1, カラム名2...) values (値1, 値2...);

  • 検索
    • select カラム名 from テーブル名 (where 条件式) (order by カラム名);

  • 更新
    • update テーブル名 set カラム名 = "更新データ";

  • 削除
    • delete from テーブル名 (where 条件式);
