gitメモ
使いそうなコマンド集
  • ローカルコンピュータで作ったリポジトリの初期化
    • git init
  • リポジトリの状態を確認
    • git status
  • ステージ領域へファイルを追加(ファイル名のところを.にするとフォルダ全部できる)
    • git add ファイル名
  • リポジトリの歴史を記録
    • git commit -m コメント
  • コミットログを確認
    • git log
  • 変更差分を確認
    • git diff
  • ブランチを一覧表示
    • git branch
  • ブランチ作成
    • git branch ブランチ名
  • ブランチの移動
    • git checkout ブランチ名
  • ブランチのマージ
    • git merge --no-ff 作業してたブランチ名
  • 歴史を戻る
    • git reset --hard コミットID
  • リモートリポジトリの登録
    • git remote add 登録名 リポジトリのパス(githubのページでコピーできるあれ)
  • リモートリポジトリへ送信
    • git push
  • リモートリポジトリを取得
    • git clone リポジトリのパス(あれ)
パターン
普段編集して上げるとき
git pull upstream master (もしローカルリポジトリが最新の状態じゃなかったら)
↓
git branch ファイル名 でブランチを分ける(ローカル環境内でmasterがバックアップとなる履歴)
↓
git checkout ブランチ名 で作ったブランチへ移動
↓
実作業が上手くいった
↓
git add ファイル名 でファイル追加
↓
git commit -m ファイル名 でコミット
↓
git checkout master でマスターへ帰る
↓
git merge --no-ff 作業してたブランチ名 で作業してたところの履歴をmasterの履歴に反映
↓
git push で自分のgithubにアップロード(ローカルからオンラインへ)
↓
github上で元々のリポジトリにプルリクエスト送信
↓
他の人にマージしてもらう(これでみんなの原本のリポジトリに自分の変更が反映)
